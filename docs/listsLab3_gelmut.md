## Как создать репозиторий

### шак 1: подготовка структуры

1. **создать папку проекта** со следующей структурой
   - файл `README.md` для описания проекта
   - папка `src/` для хранения исходного кода
   - папка `docs/` для документации
   - файл `gitignore` для исключения ненужных файлов

### шаг 2: настройка репозитория

2. **инициализировать git-репозиторий**
   `git init`
3. настройка репозитория
   добавить все файлы в staging area
   `git add .`
4. проверить статус изменений
   `git status`
5. создать первый коммит с описанием
   `it commit -m "initial commit: базовая структура проекта"`

### шаг 3: работа с удаленным репозиторием

4. настройка удаленного репозитория

```
git remote add origin <url>
git push -u origin main
```
