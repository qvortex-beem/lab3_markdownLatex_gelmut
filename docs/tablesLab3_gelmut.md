| **Функция**          | **Команда Git** | **Описание**                |
| :------------------- | :-------------- | :-------------------------- |
| Создание репозитория | **git init**    | инициализация репозитория   |
| Просмотр состояния   | **git status**  | Показывает измененные файлы |
| Отправка на GitHub   | **git push**    | Загружает коммиты на сервер |
